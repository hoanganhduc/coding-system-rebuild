# ARCHITECTURE

## Surfaces

`MANIFEST.yaml` defines ten fail-closed roots (every top-level entry under each
root must be classified or `make sync` aborts). The table combines
`.codewhale`/`.deepseek` for readability and lists `~` extras separately because
they are explicit rootless entries, not an additional fail-closed root:

|---|---|---|---|---|
| `.claude` | Claude Code | settings/mcp templates, CLAUDE.md, hooks, docs, templates, instructions, commands, agents, rules, skills (real files) | secrets.json, .credentials.json, memory, learnings, zotero/calibre config | sessions, caches, projects, data symlinks |
| `.codex` | Codex | config.toml template, AGENTS.md, agents, hooks, skills, get-shit-done | auth.json | runtime, sessions, logs (~8GB) |
| `.copilot` | Copilot CLI | config/settings/permissions, agents, skills, ide | mcp-config.json | session-state, logs |
| `.config/opencode` | OpenCode | jsonc, AGENTS.md, agents, commands, skills, package(+lock).json | ‚Äî | node_modules, opencode.db |
| `.codewhale` + `.deepseek` | CodeWhale/DeepSeek | config templates (key-redacted), AGENTS.md, mcp.json, agents, skills, tools | config.toml (full), secrets/, memory, learnings | snapshots (8.6GB), sessions |
| `.gemini` | Gemini/AntiGravity | GEMINI.md, config, settings/keybindings/skills/plugins/builtin | oauth token | brain, cache, conversations, bin |
| `.ai-agents-skills` | installer state home | ‚Äî | ‚Äî | all (regenerated by installer) |

`~/.openclaw` is **delegated**: its public sync/install belongs to the
`openclaw-bot` component (REBUILD-MANIFEST.json). Its SECRET files are still
owned by this repo's secrets manifest (single-zip principle).
Skill SKILL.md symlinks are **delegated** to the `ai-agents-skills` installer.
The `research_compute` GHA/Modal broker is **delegated** too ‚Äî its code installs to the
runtime root (not captured); only its per-install `research-compute.toml` rides the zip.

## Manifest semantics (`MANIFEST.yaml`)

- classes: `public-copy`, `public-template`, `private-archive`, `exclude-*`, `delegate`
- first-match-wins at file level, manifest order matters
- template verbs: `home-substitute` (`/home/<user>` ‚áÑ `{{ HOME }}` ‚Äî applied to
  ALL public text files), `key-redact` (named keys ‚Üí `{{ KEY }}`),
  `secret-env-split` (bashrc managed block), `emit-keys` (key names only)
- invariants enforced by the engine: symlinks never copied (recorded against
  `system/symlinks.tsv`), ELF binaries refused in public classes, nested
  `.git` dirs never captured, zero `/home/<user>` literals survive rendering,
  private paths cross-checked against the secrets manifest

## Leak scanning (3 layers, fail-closed)

1. generic patterns (token prefixes, JWT, PEM, AWS, telegram-token shape,
   40-hex with public-hash context exemptions, home paths)
2. secret KEY NAMES followed by literal values (placeholders allowed;
   `/tests/` fixture paths exempt from this layer only)
3. private denylist `~/.config/coding-system/leak-denylist.txt` (low-entropy
   personal IDs; ships in the zip; also consumed by openclaw-bot's redaction)

Canaries are constructed at runtime by `tests/leak_scan_selftest.sh` ‚Äî nothing
token-shaped is ever committed.

## Components (`components.lock`)

`name=https-url@ref`; `ref` is a commit SHA after publish, or
`LOCAL:~/path` before publish (then `external/<name>` symlinks the live
checkout). `make components LOCAL=1` forces live mode. **ai-agents-skills is the single
source of truth, cloned to `~/ai-agents-skills`** (the path the installed skill SKILL.md
symlinks resolve against, and the source Phase 8 installs the broker from); other components
live under `external/`. The umbrella owns
docker images and systemd rendering; openclaw-bot's `install.sh` is invoked
with `--skip-docker --skip-services` so there is exactly one writer per surface.

## Symlink topology

`system/symlinks.tsv` is the canonical list (verification + recreation):
`.claude/.local ‚Üí .openclaw/workspace/.local` (shared python env),
`.claude/data/{research,runs,job-queue} ‚Üí workspace data`,
plugin-skills ‚Üí npm-global openclaw dist, `agents/sandbox ‚Üí agents/main`,
`zot ‚Üí workspace zotero/zot.py`, research-skills-venv ‚Üí `~/.venvs`.
`/usr/local/bin` links (sudo) live in `system/bin/usr-local-bin.tsv`.
